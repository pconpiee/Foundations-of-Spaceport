# ISU–GSA Global Spaceport Leadership Series
## Module 1: The Strategic Site

A professional executive online course — the first in a six-module diploma program developed jointly by the **International Space University (ISU)** and the **Global Spaceport Alliance (GSA)**.

---

## Live Course

Open `index.html` in any modern browser, or host on GitHub Pages.

**GitHub Pages setup:**
1. Push this repository to GitHub
2. Go to Settings → Pages → Source: Deploy from branch → main → / (root)
3. Your course will be live at `https://[username].github.io/[repo-name]/`

## Design

- **Typography**: Cormorant Garamond (editorial display) + DM Sans (body)
- **Aesthetic**: Deep-space observatory meets executive briefing room
- **Palette**: Near-black backgrounds, gold and arctic-blue accents
- **Features**: Animated starfield, progress tracking, quiz with immediate feedback, localStorage persistence

## Technical

- Pure HTML/CSS/JavaScript — no build tools, no dependencies, no CDN required
- GitHub Pages compatible out of the box
- Progress saved in browser localStorage
- Fully responsive (desktop primary, mobile-compatible)
